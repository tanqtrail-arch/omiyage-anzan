# おみやげ算ゲーム 🎁

さくらんぼ計算 & おみやげ算の暗算トレーニングゲーム

## ゲーム内容

### 🍒 さくらんぼ計算（たし算） Lv1〜3
- レベル1: 一けた＋一けた
- レベル2: 二けた＋一けた
- レベル3: 二けた＋二けた

### 🍒 さくらんぼ計算（ひき算） Lv4〜6
- レベル4: 二けた−一けた
- レベル5: 二けた−二けた
- レベル6: 三けた−二けた

### 🎁 おみやげ算（かけ算） Lv7〜14
- レベル7〜13: 11×11 〜 19×19 の段階別練習
- タイムアタック: 全範囲60秒チャレンジ（ランキング機能付き）

## 機能
- ⭐ 3段階の星評価
- 🪙 TRAIL ALT報酬システム
- 🏆 タイムアタックランキング（週間 / 永久）
- 📖 解き方ヒント表示
- 🔥 コンボシステム
- 💬 LINE報告機能
- 📱 タブレット対応レスポンシブUI

## セットアップ

```bash
npm install
npm run dev
```

## ビルド

```bash
npm run build
```

`dist/` フォルダが生成されます。
